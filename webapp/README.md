# Queering NLP Bias!

*Experiments and Intuitions*

## Run

```bash
./webapp/run.sh
```

Auto-loads API keys from `ANTHROPIC_API_KEY` / `OPENAI_API_KEY` env vars.

## Modes

**Tree Exploration** - Compare multiple text prefixes side-by-side

**Dynamics Analysis** - Watch scores evolve as text grows

**Judge Evaluation** - Score any text against questions

## Settings

Click ⚙️ to configure:
- Provider (Anthropic/OpenAI)
- Model
- Temperature
- Judge prompt template
- API keys
